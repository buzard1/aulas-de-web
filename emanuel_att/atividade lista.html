Canetas
Instrumento para escrita com tinta.
Lápis
Instrumento para escrita ou desenho com grafite.
Cola
Material adesivo para fixar objetos.
Régua
Instrumento para medir e desenhar linhas retas.
Apontador
Usado para afiar lápis.
Marcador
Caneta para marcar textos importantes.
        Instruções:
        - Sempre pegar material devolta quando emprestar.
        - Utilize os materiais com responsabilidade.
        - Guarde os itens após o uso para evitar perdas.
        - Não usar celular nas aulas.
        - Cuidar dos materias.
        - Usar uniforme.
